# Analisis Data Penjualan E-Commers

Project ini berisi tentang analisis dataset penjualan toko online menggunakan MySQL dan Microsoft Excel untuk menemukan tren bisnis dan performa produk.

## 📊 Tentang Dataset
Dataset yang digunakan dalam project ini mencakup data transaksi dari bulan April dan Mei. File dataset yang dilampirkan adalah:
* `dataset_penjualan.xlsx` - Data mentah (raw data) yang digunakan.
* `query_analisis.sql` - Script SQL yang digunakan untuk mengolah data.

### Atribut Data (Kolom):
* `ID_Transaksi`: Kode unik setiap transaksi.
* `Tanggal`: Tanggal terjadinya transaksi.
* `Nama_Produk`: Produk yang terjual.
* `Jumlah`: Kuantitas produk yang dibeli.
* `Customer ID`: Kode unik untuk mengidentifikasi setiap pelanggan.
* `Harga Beli`: Modal awal yang dikeluarkan.
* `Harga Jual`: Total nominal yang dibayar pelanggan..

## 🚀 Langkah Analisis
1. **Data Cleaning:** Membersihkan data duplikat dan menyelaraskan format tanggal di Excel.
2. **Database Setup:** Membuat database baru di MySQL dan meng-import file CSV/Excel ke dalam tabel.
3. **Data Exploration:** Menulis query SQL untuk mencari produk terlaris dan total pendapatan bulanan.ng-mysql
